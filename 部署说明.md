# 本地后端部署指南

## 1. 安装依赖

```bash
cd C:\menu-backend
npm install
```

## 2. 启动后端服务

```bash
npm start
```

看到以下输出表示启动成功：
```
=================================
餐厅点餐后端服务已启动
本地地址: http://localhost:3000
=================================
```

## 3. 安装 Cloudflared (Cloudflare Tunnel)

### Windows 安装方式：

**方式一：使用 winget (推荐)**
```powershell
winget install cloudflare.cloudflared
```

**方式二：手动下载**
1. 访问：https://github.com/cloudflare/cloudflared/releases
2. 下载 `cloudflared-windows-amd64.exe`
3. 重命名为 `cloudflared.exe`
4. 放到 `C:\Windows\System32` 或其他 PATH 路径

## 4. 启动 Cloudflare Tunnel

```bash
cloudflared tunnel --url http://localhost:3000
```

启动后会显示类似：
```
+--------------------------------------------------------------------------------------------+
|  Your quick Tunnel has been created! Visit it at (it may take some time to be reachable):  |
|  https://xxxxx-xxxxx-xxxxx.trycloudflare.com                                              |
+--------------------------------------------------------------------------------------------+
```

复制这个 `https://xxxxx.trycloudflare.com` 地址

## 5. 修改小程序配置

打开 `C:\menu-miniprogram\app.js`，修改第 8 行：

```javascript
globalData: {
  cart: [],
  userInfo: null,
  baseUrl: 'https://xxxxx.trycloudflare.com'  // 改成你的 Cloudflare Tunnel 地址
}
```

## 6. 小程序配置合法域名

在微信公众平台 -> 开发 -> 开发管理 -> 开发设置 -> 服务器域名：

添加你的 Cloudflare Tunnel 地址到 `request合法域名`：
```
https://xxxxx.trycloudflare.com
```

## 7. 测试

1. 启动后端：`npm start`
2. 启动隧道：`cloudflared tunnel --url http://localhost:3000`
3. 修改小程序地址
4. 在微信开发者工具中预览

## 注意事项

1. Cloudflare Tunnel 免费版每次重启会生成新地址
2. 电脑休眠/关机后服务会停止
3. 建议开发测试时使用，正式上线建议部署到云服务器

## API 接口说明

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/dishes | GET | 获取菜品列表 |
| /api/dishes/:id | GET | 获取菜品详情 |
| /api/orders | POST | 创建订单 |
| /api/orders | GET | 获取订单列表 |
| /api/orders/:id/cancel | POST | 取消订单 |
| /api/orders/:id/pay | POST | 支付订单 |
