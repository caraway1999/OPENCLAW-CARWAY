# 餐厅点餐微信小程序

## 项目结构

```
menu-miniprogram/
├── app.js                    # 应用入口
├── app.json                  # 应用配置
├── app.wxss                  # 全局样式
├── project.config.json       # 项目配置
├── sitemap.json              # 站点地图
├── pages/
│   ├── index/                # 首页
│   ├── menu/                 # 菜单页
│   ├── cart/                 # 购物车页
│   └── order/                # 订单页
├── cloudfunctions/
│   ├── menu/                 # 菜单云函数
│   └── order/                # 订单云函数
└── images/                   # 图片资源目录
```

## 功能特性

1. **首页**
   - 餐厅信息展示
   - 轮播图广告
   - 快捷入口导航
   - 今日推荐菜品

2. **菜单页**
   - 分类筛选（热菜、凉菜、海鲜、汤类、主食、饮品）
   - 搜索功能
   - 加入购物车
   - 购物车快捷结算

3. **购物车页**
   - 商品管理（增减数量、删除）
   - 全选/取消全选
   - 订单备注
   - 用餐方式选择（堂食、打包、外卖）

4. **订单页**
   - 订单列表展示
   - 订单状态筛选
   - 取消订单
   - 订单支付
   - 再来一单

## 部署上线步骤

### 1. 准备工作

1. 注册微信小程序账号：https://mp.weixin.qq.com/
2. 获取 AppID（在小程序后台 -> 开发 -> 开发管理 -> 开发设置）
3. 下载微信开发者工具：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 2. 配置项目

1. 打开 `project.config.json`，将 `appid` 修改为你的 AppID
2. 打开 `app.js`，将 `env` 修改为你的云开发环境 ID

### 3. 开通云开发

1. 在微信开发者工具中打开项目
2. 点击工具栏的「云开发」按钮
3. 开通云开发服务
4. 创建云开发环境
5. 获取环境 ID

### 4. 创建数据库集合

在云开发控制台 -> 数据库中创建以下集合：

1. **dishes** - 菜品集合
2. **orders** - 订单集合

### 5. 部署云函数

1. 右键点击 `cloudfunctions/menu` 目录
2. 选择「上传并部署：云端安装依赖」
3. 对 `cloudfunctions/order` 重复上述操作

### 6. 上传图片

将菜品图片和轮播图上传到云存储，并在代码中更新图片链接。

### 7. 添加 TabBar 图标

在 `images/` 目录下添加以下图标文件：
- home.png / home-active.png（首页）
- menu.png / menu-active.png（菜单）
- cart.png / cart-active.png（购物车）
- order.png / order-active.png（订单）

图标尺寸建议：81px x 81px

### 8. 本地调试

1. 在微信开发者工具中预览
2. 测试所有功能是否正常
3. 检查云函数是否正常工作

### 9. 提交审核

1. 点击工具栏的「上传」按钮
2. 填写版本号和项目备注
3. 在小程序后台 -> 管理 -> 版本管理 中提交审核

### 10. 发布上线

审核通过后，在版本管理页面点击「发布」即可上线。

## 注意事项

1. 小程序需要企业认证才能使用支付功能
2. 个人小程序无法使用支付功能，可以改为模拟支付
3. 菜品数据需要在云数据库中手动添加或通过云函数添加
4. 建议在开发阶段使用测试号进行调试

## 菜品数据示例

在云数据库 `dishes` 集合中添加以下数据：

```json
{
  "name": "红烧肉",
  "price": 48,
  "image": "cloud://your-env-id/dishes/hongshaorou.jpg",
  "category": "热菜",
  "description": "经典红烧肉，肥而不腻，入口即化",
  "sales": 256
}
```
